Sık Sorulan Sorular

TikTok API olmadan çalışır mı?

Evet. TikTok API olmadan videolar otomatik olarak output/Videolar/ klasörüne kaydedilir, manuel yükleme yapabilirsiniz. Diğer platformlar (YouTube, Instagram, Facebook) bağımsız çalışır.

Hangi API'ler gerekiyor?

Groq API (ücretsiz), Pexels API (ücretsiz). TikTok/YouTube/Instagram API'leri opsiyoneldir — sadece otomatik yükleme için gereklidir.

Lisans kaç cihazda çalışır?

Her lisans 1 cihaza özeldir (HWID bazlı). Cihaz değişikliğinde destek hattımızdan yeni lisans talep edebilirsiniz.

Günde kaç video üretebilir?

Sınırsız. Groq API'nin ücretsiz kotasına göre saatte 10-50 video üretimi mümkündür. Zamanlayıcı ayarları tamamen size bağlıdır.

FFmpeg kurulu olmalı mı?

Hayır. ViralBot Pro, FFmpeg'i otomatik olarak indirir ve yapılandırır. Ekstra kurulum gerekmez.
